# ポートフォリオ
https://kskwes.github.io/my-portfolio-v2/

## 環境
- Nuxt.js
- Bulma

### Github Pagesにデプロイする方法
1. 変更を commit する
2. 静的アプリケーションを生成 
```
npm run generate
```
3. GitHub Pages にデプロイ 
```
npm run deploy
```
